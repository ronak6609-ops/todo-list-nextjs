# Daily Routine Tracker

A modern and simple Daily Routine Tracker built using Next.js and React.  
This application helps users track daily habits, maintain streaks, and stay consistent.

## 👨‍💻 Author
- GitHub: https://github.com/ronak6609-ops


## 🚀 Features

- Track daily routines easily
- Monitor progress with visual indicators
- Maintain streaks for consistency
- Organize tasks (Morning, Afternoon, Evening)
- Add and edit routines
- Clean and responsive UI
- Dark mode support


## 🛠️ Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
